# Mobile App Feature Gaps

Features present in the web app but missing from the mobile app.

## ~~Messaging~~ (partial)
- ~~**Message editing** — edit your own messages~~ ✅
- ~~**Message deletion** — delete your own messages~~ ✅
- ~~**Reactions** — add/remove emoji reactions~~ ✅
- ~~**Rich text / markdown** — bold, italic, links, lists, code blocks~~ ✅
- ~~**@mentions** with autocomplete suggestions~~ ✅
- ~~**File/image uploads** and attachment previews~~ ✅
- ~~**Typing indicators**~~ ✅

## ~~Threads~~ ✅
- ~~**Thread panel** — view and reply in threads~~
- ~~**Edit/delete in threads**~~ ✅
- ~~**Load more replies**~~ ✅

## Search
- **Full-text message search**
- Filter by channel, user, date range
- Navigate to message from search result

## Huddles (Audio/Video)
- **Start/join huddles** in channels or DMs
- Mute/unmute, camera toggle, screen sharing
- Device selector, participant list, video grid
- LiveKit integration — entirely absent on mobile

## Channels
- **Create channels**
- **Private channels** support
- **Channel members dialog** — view/add/remove members
- **Channel search**

## Direct Messages
- **Create new DMs** from a user picker dialog

## User Profiles
- **User profile sidebar** — view profile, role, presence, initiate DM
- **Profile editing** — display name, avatar

## Workspace Management
- **Create workspaces**
- **Workspace settings** dialog
- **Workspace switcher** in sidebar

## Invitations
- **Generate invite links**
- **Accept invites** via link/code

## Notifications
- **Push notification settings** — toggle on/off, sound control
- **Notification permission** handling

## Layout & UX
- **Mark as read** UI beyond viewing
- **Theme toggle UI** (mobile detects system theme but has no manual toggle)
